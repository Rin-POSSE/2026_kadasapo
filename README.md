## display

display プロパティの違いを確認する課題です。  
`inline`、`block`、`inline-block` の挙動を比べながら修正しましょう。

### question-1

リンクにの幅指定と中央寄せが効くボタンにしましょう。

確認するファイル:

- `display/question-1.html`

練習内容:

- `a` タグの初期状態
- `inline` 要素に `width` が効かない理由
- `block` を追加したときの変化

### question-2

改行されている要素を横並びにしましょう。  


確認するファイル:

- `display/question-2.html`

練習内容:

- `div` と `span` の違い
- `block` と `inline` の違い
- 横並びになる条件

### question-3

縦に並んでいるボタンを横並びにしましょう。

確認するファイル:

- `display/question-3.html`

練習内容:

- `block` による縦並び
- `inline-block` による横並び
- 横並びしながら `width` や `padding` を効かせる方法
